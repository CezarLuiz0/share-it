share-it

Modern share. No dependencies. Just JS. Just 4kb.

Use share-it with one click and share your data easily!

Supported Social Networks

  • Facebook
  • Twitter
  • Google+
  • Pinterest
  • Linked In
  • Buffer
  • Digg
  • StumbleUpon
  • Tumblr
  • Reddit
  • Evernote
  • Wordpress
  • Pocket
  • Whatsapp

Install

Using bower:

bower install --save share-it

Usage

var share = new ShareIt({
  specs: {
    width: 720,
    height: 300
  }
});

share.init();

After initialization, just put data-share attribute in your target button or whatever element.

<button
  class="btn"
  data-share
  data-share-facebook
  data-share-facebook-url="http://cezarlz.github.io/share-it"
>
  Share on Facebook
</button>

<button
  class="btn"
  data-share
  data-share-whatsapp
  data-share-whatsapp-text="Hi guys, like share-it!"
>
  Share on Whatsapp (only mobile)
</button>

API

Init params

Property Type Default Required?
specs.width number 550 No
specs.height number 350 No
target string '_blank' No
newTab boolean false No

In specs, you can use another values. See more here.

For example:

var share = new ShareIt({
  specs: {
    left: 20,
    top: 20,
    menubar: 'no',
    titlebar: 'yes',
    resizable: 'no'
  }
});

share.init();
